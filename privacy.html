අභ්‍යන්තර දත්ත පමණක්

මෙම වෙබ් පද්ධතිය DSI ආයතනයේ අභ්‍යන්තර ප්‍රවාහන කළමනාකරණය සඳහා පමණක් භාවිතා වේ. Company internal data only — බාහිර මහජන සේවාවක් නොවේ.

අපි එකතු කරන දත්ත

  • Login සඳහා: email, password hash (සර්වරයේ), role, status.
  • සේවකයන් සඳහා: emp no, නම, department, default route/sub-route, active status.
  • දිනපතා ඉල්ලීම්/අනුමත කිරීම: request records, route/sub-route grouping, vehicle assignment.
  • තාක්ෂණික: browser localStorage (token/role/email) — login session පවත්වාගෙන යාමට.

දත්ත භාවිතා කරන්නේ මොකටද?

  • දිනපතා ප්‍රවාහන ඉල්ලීම් සකස් කිරීම සහ අනුමත කිරීම (HOD → Admin → TA → HR).
  • රූට්/සබ්-රූට් අනුව සේවකයින් වෙන් කරවා වාහන පවරීම සහ capacity validate කිරීම.
  • Report (Route-wise / Vehicle-wise) සාදමින් HR final approval පසු download කිරීම.

දත්ත ප්‍රවේශය සහ බෙදාගැනීම

  • Role-based access: Admin/HOD/TA/HR/EMP roles අනුවම screens & API access සීමා කර ඇත.
  • අභ්‍යන්තර අවශ්‍යතාවයට පමණක් — තෙවන පාර්ශ්වයන්ට (third parties) අලෙවි/බෙදා නොදේ.

දත්ත රඳවා තබාගැනීම

ආයතනික අවශ්‍යතාව (audit/report) අනුව දත්ත රඳවා තබාගනී. අවශ්‍ය නොවන පරණ දත්ත පසුව සමාගම විසින් ඉවත් කළ හැක.

ඔබගේ වගකීම

  • ඔබේ login විස්තර වෙන අයට නොදෙන්න.
  • Public/Shared device වලින් login වුවහොත් logout කරන්න.

සම්බන්ධ වීමට

ඕනෑම පෞද්ගලිකත්ව ගැටළුවක්/පැමිණිල්ලක් සඳහා: contract@dsitransport.top
